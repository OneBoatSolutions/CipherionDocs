## Setup Steps

Follow these detailed guides to complete your Cipherion setup:

### 1. Account Setup

Create your Cipherion account and configure your organization settings. This is the foundation for all your encryption projects.

**What you'll learn:**
- How to create a Cipherion account
- Organization configuration
- Team member management
- Initial security settings

**Estimated time:** 5 minutes

[Start Account Setup →](/get-started/account-setup)

---

### 2. Creating a Project

Set up your first project and configure basic security policies. Projects help you organize and manage different encryption environments.

**What you'll learn:**
- Project creation process
- Security policy configuration
- Environment setup
- Project settings management

**Estimated time:** 10 minutes

[Start Creating a Project →](/get-started/creating-project)

---

### 3. Setting up API Key

Generate and configure API keys for your applications. API keys authenticate your applications with Cipherion's encryption services.

**What you'll learn:**
- API key generation
- Key permissions and scopes
- Secure key storage
- Key rotation practices

**Estimated time:** 5 minutes

[Start Setting up API Key →](/get-started/api-key)

---

### 4. Key Generation by Passphrase

Learn how to generate encryption keys using secure passphrases. This is a critical step for maintaining data security.

**What you'll learn:**
- Passphrase requirements
- Key generation process
- Secure passphrase storage
- Best practices

**Estimated time:** 10 minutes

[Start Key Generation →](/get-started/key-generation)

---

### 5. Changing Passphrase

Update your passphrase and maintain security best practices. Regular passphrase updates are essential for security.

**What you'll learn:**
- Passphrase rotation process
- Data re-encryption
- Security considerations
- Recovery procedures

**Estimated time:** 5 minutes

[Start Changing Passphrase →](/get-started/changing-passphrase)

---
